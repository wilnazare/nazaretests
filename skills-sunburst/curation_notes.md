# Skills Sunburst Curation Notes

Current viewer-facing curation notes. Git history tracks earlier versions.

Current conventions:
- Selected subskills show percentage as "x% of job adverts".
- Heading-level and top-subskill counts use "job adverts" rather than acronym shorthand.
- Weighted score is kept in the CSVs for audit/debugging, not shown in the viewer.
- Common phrases are canonical phrase families, not raw extracted n-gram variants.

Counts still use the same de-duplicated 100-job-advert corpus baseline.
